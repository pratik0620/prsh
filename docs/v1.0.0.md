# prsh v1.0.0

Released: 15 June 2026

## What's in this release

The first working version of prsh.

## Features

- REPL loop
- External command execution via fork() and execvp()
- Built-in commands: cd, clear, exit
- Single pipe support (|)
- I/O Redirection: >, >>, <,
- Here documents (<<)

## What I learned building this

- How fork() creates a child process and why exec() needs it
- File descriptors 0, 1, 2 and how dup2() rewires them
- Why cd must be a built-in — child processes can't change
  the parent's working directory
- How a pipe connects two processes through a kernel buffer
- Why closing unused pipe ends is critical — grep hangs
  forever if the write end stays open

## Known issues at release

- Only single pipe supported
- No signal handling — Ctrl+C kills the shell
- No command history
- No background processes
