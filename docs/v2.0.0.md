# prsh v2.0.0

Released: 22 June 2026

## What's new

### Signal Handling

- Ctrl+C (SIGINT) kills the foreground child but not the shell
- Ctrl+D exits cleanly
- Child processes reset to default signal handlers before exec

### Background Processes

- Commands ending with & run in the background
- Shell reaps finished background processes at each prompt

### Command History

- Persistent history saved to ~/.prsh_history
- !! repeats last command
- !n runs command number n
- !-n runs nth most recent command
- history built-in lists all commands
- history -c clears history

### Environment Variables

- $VAR expansion in commands
- export VAR=value sets environment variables
- unset VAR removes them

### Multiple Pipes

- ls | grep .cpp | wc -l now works
- N commands need N-1 pipes — all created before forking

### Pipes + Redirection Combined

- ls | grep cpp > out.txt now works correctly
- Redirection handled per-command inside the pipe loop

## Architecture change

- Moved from single main.cpp to multi-file structure
- Separate modules for parser, pipe, redirect, builtin,
  executor, signal_handler, history, utils

## Bug fixes from v1.0.0

- Fixed variable shadowing in pipe loop
- Fixed file descriptor leak on fork failure
- Fixed Ctrl+D causing infinite loop

## Known limitations

- No arrow key navigation
- No tab completion
- No quoted string support
